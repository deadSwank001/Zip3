## 🚀 The Mission
Project YellowPages is a high-fidelity data pipeline designed to bridge the gap between cutting-edge technology and human networking. We identify the companies leading the human-tech frontier (curated from the Wired Magazine tech review team) and enrich them with verified contact data using advanced AI.

### 🛠️ The Pipeline
1. **B-raz Index (Core):** The initial list of market leaders.
2. **B-raz Index (2):** AI-enriched contact details, HQ locations, and direct emails.
3. **B-raz Orgo:** The final, structured directory optimized for CRM integration.
4. **BickleRick Legacy Data:** A cleaned archive of X (Twitter) user data from 2020—finding the "un-findable" users.
